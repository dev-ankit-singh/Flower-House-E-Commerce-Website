# 🌸 Flower House - E-Commerce Website

A modern full-stack e-commerce website for selling flowers, flower pots, decorative planters, and gardening products. The application provides a clean, responsive, and user-friendly shopping experience.

## ⚙️ Installation

Clone the repository

bash
git clone https://github.com/dev-ankit-singh/Flower-House-E-Commerce-Website.git


Go to project

bash
cd Flower-House-E-Commerce-Website


Install dependencies

bash
npm install


Create a `.env` file

env
MONGODB_URI=your_mongodb_connection_string


Start the application

bash
npm start
